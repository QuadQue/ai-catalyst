# ai-catalyst

AI Catalyst is an AI agent builder and management platform.

## Features
- AI Agent Management
- Task Automation
- Performance Analytics
- Team Collaboration
- Real-time Monitoring

## Tech Stack
- Next.js 13
- TypeScript
- Tailwind CSS
- Radix UI
- Recharts

## Getting Started

### Prerequisites
- Node.js (v14 or later)
- npm (v6 or later)
